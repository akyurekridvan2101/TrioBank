# Sistem Mimarisi (C4 Model)

Bu dizin, TrioBank mimarisini açıklayan C4 model diyagramlarını içerir.

## 1. Sistem Bağlam Diyagramı (System Context)
Sistem Bağlam diyagramı, TrioBank'a ve içinde bulunduğu ortama kuş bakışı bir bakış sağlar. Yazılım sistemini bir kara kutu olarak gösterir ve kullanıcılar ile dış sistemlerle olan etkileşimlerini belirtir.

![System Context](context.svg)

## 2. Konteyner Diyagramı (Container Diagram)
Konteyner diyagramı, sistemi oluşturan konteynerleri (uygulamalar, veri depoları, mikroservisler) göstermek için yazılım sistemine yakından bakar. Yazılım mimarisinin genel şeklini ve sorumlulukların nasıl dağıtıldığını gösterir.

![Container Diagram](container.svg)
